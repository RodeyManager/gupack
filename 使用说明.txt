1、首次冲svn上checkout后，需要在本目录下执行

   npm install
   或
   npm install --no-bin-links

   安装gulp所需要的插件
----------------------------------------------------------------
2、projects.js 为项目列表文件

   执行命令时可以指定项目文件
   gulp -p mall --projectFile ../projects.js
   
   列出所有项目
   gulp -l 或 gulp --list
----------------------------------------------------------------
3、gulpfile.js 为gulp任务执行默认文件

   注意： 执行gulp命令是可以指定gulpfile.js文件

   gulp --gulpfile gulpfile_test.js
   或者:
   gulp --gulpfile ../cmbx/mall/src/gulpfile.js
----------------------------------------------------------------
4、每个项目的自动配置文件可以放到该项目的src目录下

   如：../cmbx/mall/src/gulp-config.js

5、执行命令时指定项目，gulp将自动以指定的项目进行编译
   gulp -p mall
----------------------------------------------------------------
6、如果自己写了任务，需要另外的插件，可以自行安装，
   但注意请使用

      npm i -D 插件名  (或者： npm install --save-dev 插件名)

   进行安装，以便后续使用者可以安装到你使用的插件
----------------------------------------------------------------
7、gulp-config.js为公共的gulp任务执行配置文件（不要去修改），
   推荐你在copy到当前项目的src下，然后根据项目具体进行修改
